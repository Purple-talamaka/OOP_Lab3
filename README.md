  ## Описание

  GUI-приложение на C# (Windows Forms, .NET 8).
  При клике мышью на форме создаются круги. Круги можно выделять и удалять.

  ## Функциональность

  - Клик ЛКМ на пустом месте — создать новый круг
  - Клик ЛКМ на круге — выделить его
  - Ctrl + ЛКМ — добавить к выделению / снять выделение
  - Delete — удалить все выделенные круги

  ## Структура проекта

  - `CCircle.cs` — класс круга (координаты, радиус, отрисовка, hit-тест)
  - `CircleStorage.cs` — собственный контейнер для хранения кругов
  - `Form1.cs` — главная форма, обработка событий
